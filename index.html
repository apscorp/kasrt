Konfigurasi Aplikasi

Masukkan URL Apps Script dari Google Sheets Anda.

Cara Setup (sekali saja):
  1. Buka Google Sheets baru, beri nama "Transaksi" di tab sheet
  2. Extensions → Apps Script → paste isi file apps-script.js
  3. Ganti SPREADSHEET_ID dengan ID spreadsheet Anda
  4. Deploy → New Deployment → Web App → Anyone → Deploy
  5. Copy URL deployment, paste di atas
  6. Klik Simpan → bagikan LINK yang muncul ke semua warga
