Konfigurasi

Hubungkan ke Google Sheets sebagai database.
Cara setup (sekali saja):
  1. Buka Google Sheets baru
  2. Extensions → Apps Script → paste kode dari apps-script.js
  3. Ganti SPREADSHEET_ID dengan ID spreadsheet Anda
  4. Deploy → New Deployment → Web App → Anyone → Deploy
  5. Copy URL deployment → paste di atas → klik Simpan
  6. Salin link yang muncul → bagikan ke semua warga
Tersimpan! Bagikan link ini ke warga:
Warga buka link sekali → langsung terhubung di browser mereka.
