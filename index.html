⚠️ Konfigurasi Firebase Diperlukan

Anda belum mengisi konfigurasi Firebase. Data saat ini hanya tersimpan di browser lokal. Ikuti langkah berikut agar data sinkron ke semua warga secara real-time:

  1. Buka console.firebase.google.com → buat project baru (gratis)
  2. Klik Firestore Database → Create database → pilih mode Test mode
  3. Di menu kiri klik ⚙️ Project settings → scroll ke bawah → salin konfigurasi firebaseConfig
  4. Buka file index.html ini, cari bagian 🔥 GANTI DENGAN KONFIGURASI FIREBASE ANDA, dan tempel nilai yang disalin
  5. Upload ulang ke GitHub Pages — selesai! 🎉
ℹ️ Seluruh warga dapat memantau kas ini secara langsung. Transparansi untuk kepercayaan bersama.
