⚙️ Konfigurasi Google Sheets

Hubungkan aplikasi ini ke Google Sheets sebagai database.

📋 Cara Setup (dilakukan SEKALI oleh Bendahara):
  1. Buka Google Sheets, buat spreadsheet baru
  2. Buka menu Extensions → Apps Script, paste kode dari apps-script.js
  3. Ganti SPREADSHEET_ID dengan ID spreadsheet Anda
  4. Klik Deploy → New Deployment → Web App (Anyone can access)
  5. Copy URL yang muncul → paste di field atas
  6. Klik "Simpan & Unduh File Baru" → file keuangan-rt2.html akan terunduh otomatis
  7. Bagikan file keuangan-rt2.html itu ke semua warga — langsung bisa dipakai tanpa konfigurasi!
💡 Kenapa unduh file baru? Konfigurasi tertanam langsung di dalam file HTML, sehingga semua warga yang membuka file tersebut otomatis terhubung ke data yang sama — tanpa perlu setting apapun.
